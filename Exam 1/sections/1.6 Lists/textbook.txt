CSS list-style-type
The CSS property list-style-type provides the ability to change the bullet used in an
unordered list and offers more numbering options in an ordered list.
Example
<ul style="list-style-type:square">
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Nested lists
A nested list is a list within a list item of another list. A nested list displays indented inside the outer
list. A nested unordered list uses a different bullet style, and a nested ordered list restarts at 1. I PARTICIPATION
